# Cointegration-for-Pairs-Trading


"Cointegration is a useful econometric tool for identifying assets which share a common equilibrium. Cointegrated pairs trading is a trading strategy which attempts to take a profit when cointegrated assets depart from their equilibrium." - https://www.sciencedirect.com/science/article/pii/S0377042715003234

---

Goal: 
1. Given a set of time series for different assets, find out if they  are cointegrated.
2. How to trade on the pair to generate a profit.
